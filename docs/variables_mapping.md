## Template variables

- slide_title (string)
- bullet_1 (string)
- bullet_2 (string)
- bullet_3 (string)
- slide_number (number)
- total_slides (number)
- brand_primary (hex color)
- brand_accent (hex color)
- brand_bg (hex color)
- font_family (string)
- logo_url (url)
- brand_tagline (string)
